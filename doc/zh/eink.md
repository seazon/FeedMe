# 适配电子墨水屏
- 1. `设置` - `界面` - `主题` - 选择`固定主题`，设置为`电子墨水屏` **颜色纯黑纯白处理，可以更好的显示**
- 2. `设置` - `操纵` - 开启`分动作条` **将动作条置于底部，可以更好的操作菜单**
- 3. `设置` - `操纵` - 开启`音量键`，并按个人喜好设置`翻页模式`和`翻转百分比` **使用音量键进行翻页操作**
- 4. 进入列表页 - 长按顶部标题栏 - 弹出`编辑菜单` - 将默认禁用的`返回`、`向上`、`向下`上移至顶部，保存 **使用点按菜单方式来代替划动手势**
- 5. 进入文章页 - 长按顶部标题栏 - 弹出`编辑菜单` - 将默认禁用的`返回`、`向上`、`向下`上移至顶部，保存 **使用点按菜单方式来代替划动手势**

更多关于自定义`动作条`，查看[这里](https://github.com/seazon/FeedMe/blob/main/doc/zh/customize_menus.md)
