FeedMe is an reader client application for RSS services:
- Feedbin
- Feedly
- Fever API
- Folo
- Tiny Tiny RSS
- Google Reader API
  - Bazqux
  - FreshRSS
  - InoReader
  - The Older Reader

This is the documentation of FeedMe.
